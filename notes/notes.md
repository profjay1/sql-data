Mistakes made
# Creating table without specifying the column names and data types.
# Placing the WHERE clause after the ORDER BY clause.
# Using the column name of one table in another table.

Knowledge learnt
# Proper use of the ORDER BY clause followed by ASC or DESC after the column name for sorting purpose.
# Use of LIMIT to specify the amount of records needed.
# Use of <> instead of != as the standard SQL symbol for "not equal to"
